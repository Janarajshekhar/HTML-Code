HTML Text-Level Elements

  1. The following elements embolden text :

    • this is bold text
    • this is strong text

  2. These elements italicise text :

    • this is a citation
    • this is a definition
    • this text is emphasised
    • this text is in italics
    • this text is a variable

  3. These elements render text in a fixed-width font :

    • this is a line of code
    • this is a keyboard input
    • this text is sample output

  4. Examples of supercripted and subscripted text :

    • 210 is equivalent to 1,024
    • 6410 is equivalent to the binary number 1000000

  5. We can mark text, and mark it read backwards :

    • this text is marked (highlighted)
    • !sdrawkcab lla si txet siht

  6. We can even quote great works of literature :

    • The boy stood on the burning deck...
